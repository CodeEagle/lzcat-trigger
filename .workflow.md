# 懒猫微服移植项目集合

本仓库用于收集和移植 GitHub 上优秀开源项目到懒猫微服（LazyCat）应用商店。

## 目录结构

```
.
├── README.md           # 本说明文档
├── _template/          # 项目模板，用于创建新的移植项目
└── <project-name>/     # 各个移植项目
    ├── README.md           # 项目说明
    ├── lzc-manifest.yml    # 懒猫微服应用清单
    ├── lzc-build.yml       # 构建配置
    ├── icon.png            # 应用图标
    └── .github/workflows/  # GitHub Actions 工作流
```

> [!IMPORTANT]
> **Icon 规范**：`icon.png` 文件大小不得超过 **200KB**，建议使用 512x512 像素的 PNG 格式图片。

## 如何添加新项目

### 1. 创建项目目录

```bash
# 复制模板
cp -r _template NewProject
cd NewProject
```

修改以下文件：
- `lzc-manifest.yml` - 应用配置
- `.github/workflows/update-image.yml` - 上游镜像信息
- `README.md` - 项目说明
- `icon.png` - 应用图标

### 2. 创建 GitHub 仓库

```bash
# 确保 gh 登录的是 CodeEagle 账户
gh auth switch --user CodeEagle

# 初始化 git
git init && git add . && git commit -m "Initial commit"

# 创建仓库并推送
gh repo create NewProject --public --source=. --push --description "项目描述"
```

### 3. 配置 Secrets

```bash
# 设置 GH_TOKEN（导出本地 GitHub Token）
# 从 gh auth status 获取当前登录的 token
GH_TOKEN=$(gh auth token)
echo "$GH_TOKEN" | gh secret set GH_TOKEN --repo CodeEagle/NewProject

# 设置 LZC_CLI_TOKEN（从本地懒猫配置读取）
LZC_TOKEN=$(cat ~/.config/lazycat/box-config.json | python3 -c "import sys,json; d=json.load(sys.stdin); print(d.get('token',''))")
echo "$LZC_TOKEN" | gh secret set LZC_CLI_TOKEN --repo CodeEagle/NewProject
```

### 4. 更新本仓库 README

在已移植项目列表中添加新项目。
